The tarball in this directory contains the meshes from the ModelNet40 dataset sampled into 2000-point pointclouds.



The original dataset can be found here.

If you use this dataset for your work, please credit the authors:

Wu, Zhirong, et al. "3d shapenets: A deep representation for volumetric shapes." Proceedings of the IEEE conference on computer vision and pattern recognition. 2015.
